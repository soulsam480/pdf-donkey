## PDF Donkey

PDF monkey but open-souce.

Purely hobby project. Don't know if it'll make it to a real server or not.

### Stack

api:
- Node v14.x
- Express
- SQLite (Will be postgres if deployed)
- Typeorm
- Typescript
- Route Controllers
- Auth (not finalized yet)

app:
- React (may be)
